# PowerShell Infrastructure Projects

This repository is used to learn and practice PowerShell with a focus on
hardware diagnostics, system health checks, and infrastructure automation.

## Purpose
The goal of this repository is to build practical, job-ready skills for
data center technician, server technician, and infrastructure roles.

## Focus Areas
- Hardware and system health checks
- Disk and memory monitoring
- Automation of repetitive infrastructure tasks
- Cross-platform PowerShell (macOS, Linux, Windows)

## Repository Structure
- `scripts/` – PowerShell automation scripts
- `docs/` – Documentation and walkthroughs for projects
- `notes/` – Learning notes and command references

## Environment
- macOS
- PowerShell 7
- Git and GitHub
